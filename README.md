# CarneCerta IoT

Sistema inteligente de monitoramento de câmaras frias utilizando ESP32.

## Funcionalidades

- Monitoramento de temperatura
- Controle de porta
- Alerta visual (LEDs)
- Alerta sonoro (Buzzer)
- Teste manual do sistema

## Tecnologias

- ESP32
- Wokwi
- C++
- IoT
